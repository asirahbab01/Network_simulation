Router2#
en
conf t
int s0/1/0
ip addr 192.168.50.2 255.255.255.0
no shut
ex
int s0/2/1
ip addr 192.168.40.2 255.255.255.0
no shut
ex

/*EIGRP routing implemented*/
route eigrp 6
network 192.168.40.0 0.0.0.255        
network 192.168.50.0 0.0.0.255
ex
do show ip route


N.B: 6 is the Autonomous Number(AS). All routers in the network topology must use the same AS number to share routing information. We can replace 6 with any number from 1 to 65535, as long as it matches across all routers.
